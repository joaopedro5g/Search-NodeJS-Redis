### SITEMA DE BUSCA

## PARA PODER UTILIZAR ESSE SISTEMA TERA QUE TER REDIS INSTALADO COM OU SEM DOCKER

Rode `docker pull redis` e depois `docker run -p 6379:6379 -d redis` na sua maquina para que o Redis inicie no **Docker**
